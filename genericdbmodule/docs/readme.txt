This a module from XoopsCube called XOOPS Generic DB and adjusted for XOOPS 2.5.0. It's an enhanced dummy/ghost module where the user has the capabilities to create form and display submitted data and user/admin can do all the basic database CRUD (add, read, update, delete) operations. It also has its own blocks/comments etc

The main features are as follows.
- Database field (column) can be increased or decreased in the control panel.
- can be duplicated at the time of installation. Module name can be determined freely by changing directory name
- You can install multiple modules in the XOOPS database.
- Search, event notification (add, update, delete) comments, a block display.
